### JWTAngularNode

Solução dividida em 2 partes:

### Client: Angular

Client Angular roda na porta 4200.

Para executar a aplicação utilizar os seguintes comandos:

```
cd ./Client
npm i
ng serve
```

### Server: NodeJS

Server Node roda na porta 3000.

Para executar a aplicação utilizar os seguintes comandos:

```
cd ./Server
npm i
node app.js
```
